# 깃헙에 올리는 방법.
## 먼저 git add 파일명
## 다음으로 git commit -m "수정된 사항 짧게 설명"
## 마지막으로 git push 링크 이름(이 경우에는 origin) 브랜치명(이 경우에는 master)
##
## 전역 설정 정보 조회인 git config --list 후 나가기를 하기 위해선 Q키를 누릅니다
## 저장소별 설정 정보 조회는 git config - -list 입니다.
## 
# 전역 사용자명/이메일 구성하기
## git config - -global user.name “Your name”
## git config - -global user.email “Your email address”
#
# 저장소별 사용자명/이메일 구성하기 (해당 저장소 디렉터리로 이동후)
## git config - -user.name “Your name”
## git config - -user.email “Your email address”
##
# 새로운 저장소 초기화하기
## git init
